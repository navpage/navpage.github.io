    codeblock languages¶

    In order to have codeblocks show syntax highlights note the language like: 

        ``` yaml
    theme:
    features:
        - content.code.annotate # (1)
    ```
    Some of the popular ones are: - html - py - md - css - js

    Adding annotations¶

    Code annotations can be placed anywhere in a code block where a comment for the language of the block can be placed, e.g. for JavaScript in // ... and /* ... */, for YAML in # ..., etc.1:

    Code block with annotation
    ``` yaml
theme:
  features:
    - content.code.annotate # (1)
```
